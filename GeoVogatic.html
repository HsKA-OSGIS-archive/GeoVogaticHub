Introduction

This help file was written to show users the procedure, options and capabilities of the assisted styling application in Open Layers 3, created by Geovogatic.

How it works

A new button/layer appears whenever a file is dragged and dropped over the interface. Clicking on this layer opens a pop-up menu with the designated styling options for that layer depending on geometry type. This application supports point, line and polygon geometry types.

Below are the geometries and the styling options (what can be changed) associated to each of them.

Point : Size, shape, fill colour and transparency
Line : Thickness, pattern, colour and transparency
Polygon : Border thickness, border colour, fill colour, fill pattern and transparency

Important Note

Only default OL3 geographic file formats are supported. Information about these formats can be found on the OL3 website.
